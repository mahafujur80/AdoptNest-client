# 🐾 Pet Adoption Platform (AdoptNest)

## 📖 Purpose
A full-stack pet adoption web application where users can discover pets, post pets for adoption, and manage adoption requests through a secure dashboard system.

---

## 🌐 Live URL
https://adopt-nest-client.vercel.app/

---

## ✨ Features

- 🔍 Browse all pets with search, filter, and sorting functionality
- 🔐 Secure authentication (Email/Password + Google Login)
- ➕ Add, update, and delete pet listings (private routes)
- 📩 Adoption request system with approve/reject workflow
- 📋 User dashboard for managing listings and requests
- 🐶 Pet details page with full information and adoption form
- 📱 Fully responsive and modern UI design
